RecurrenceSequenceElement and RecurrenceSequenceRingÔÉÅ

This class provides common functionalities for rings of sequences defined by linear recurrence relations. The structure of the rings is given by:

  • CommutativeAlgebra
    • rec_sequences.RecurrenceSequenceRing
      • rec_sequences.DFiniteSequenceRing

        • rec_sequences.CFiniteSequenceRing

      • rec_sequences.DifferenceDefinableSequenceRing

        • rec_sequences.C2FiniteSequenceRing

So all rings are subclasses of a commutative algebra over a base field which is the field where all terms of the sequences live.

Return whether self is the zero sequence \(0,0,0,\dots\). This is the case iff all the initial values are \(0\) or None.
